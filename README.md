# 📖 About

This is a **Quiz Game** developed using **C#** in **Visual Studio 2022**.

The application asks multiple-choice questions, checks the selected answer, keeps track of the player's score, and displays the final result after the quiz.

---

# ✨ Features

- ✅ Multiple Choice Questions
- ✅ Score Calculation
- ✅ Easy-to-use Interface
- ✅ Instant Answer Validation
- ✅ Final Result Display
- ✅ Simple and Clean Design

# 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/kajalpalve2007/Quiz-game.git
```

2. Open the project in **Visual Studio 2022**

3. Press **F5** to run the application.

---

# 🎯 Gameplay

- Start the quiz.
- Read each question carefully.
- Select the correct answer.
- Submit your answer.
- Check your final score.
